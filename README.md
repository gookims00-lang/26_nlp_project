# 🚀 The Edge Pattern Hunter: Semantic Compression & Discovery

소형 언어 모델(SLM)을 활용한 엣지 환경에서의 고효율 데이터 압축 및 숨겨진 패턴 발견 프로젝트

## 📌 Motivation (동기)
엣지 환경의 **'약한 생성기(Weak Generator)'**는 대량 데이터 처리와 복잡한 에이전트 구조에서 성능이 저하되거나 분석에 실패합니다. 본 프로젝트는 이를 해결하기 위한 '데이터 중심'의 접근법을 테스트

## 💡 Novel Hypothesis (가설)
> "생성기의 성능이 낮을수록 복잡한 에이전트 로직보다, 데이터 자체를 모델 친화적인 서사(Narrative)로 변환하는 **의미론적 압축(Semantic Compression)**이 분석 성능을 더 일관되게 보장한다."

## 🛠 Method (방법론: Edge Pattern Hunter)
1. **의미론적 압축**: 원시 수치 데이터를 일별 서사 로그로 변환하여 데이터 양절감
2. **인과 패턴 발견**: 숫자가 아닌 '사건' 기반의 로그를 통해 SLM이 데이터 사이의 숨겨진 시차(Time Lag) 인과관계를 발견하도록 유도

## 📊 Experiment (실험 및 결과)
- **Baseline**: 원시 수치 데이터 나열 방식
- **Proposed**: The Edge Pattern Hunter (의미론적 압축 로그)
- **Key Findings (60-Day Full Analytics)**: 
  - **입력 효율성**: Proposed 방식은 Baseline 대비 **1.6배의 토큰 절감**을 달성
  - **추론 속도**: 처리 시간 **1.6배 단축** (18.36초 -> 11.76초)
  - **패턴 분석**: 약한 생성기(Qwen3-8B) 환경에서 Baseline은 일반적 상관 관계만 발견했으나, Proposed 방식은 **'X일의 시차'**라는 구체적 인과 패턴을 정확히 포착
